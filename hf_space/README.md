## Ollive OSS Assistant

This Space is the OSS half of an AI assistant evaluation challenge built for Ollive AI.
It runs Qwen2.5-0.5B-Instruct locally on CPU via the Hugging Face transformers library.

Full project source: https://github.com/PLACEHOLDER/ollive-assistant-eval (will be updated once the repo is public)

Note: this is a 0.5B parameter model and will hallucinate or refuse incorrectly on some inputs.
